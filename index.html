tildenest.space

Tildenest.space is a site inspired by tilde communities. It's a computer meant for sharing, a kind of public-access unix system. We are a community that exists on a linux server.

Stuff we like:

  • Free and open-source software
  • Minimalist
  • Suckless
  • Retrocomputing
  • Socializing
  • Peer education
Our ground rules:
  • Don't host illegal content.
  • This server is not to be used for commercial purposes.
  • Respect other users as you would want to be respected.

You have two options to join:

  • Push public key to github
  • Email me user name and public key
